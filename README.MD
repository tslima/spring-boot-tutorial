# Spring Boot Tutorial

Nesse repositório irei depositar uma série de tutoriais de como estruturar uma aplicação simples em Spring-Boot. Começando desde o princípio, instalação do java e configuração do ambiente de desenvolvimento, até assuntos um pouco mais avançados como modularização da aplicação, migração de banco de dados e criação de um container.

Em conjunto iremos desenvolver a aplicação **WebCarros** para a nossa concessionária.
